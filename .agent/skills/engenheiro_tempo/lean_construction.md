# Skill: Lean Construction

## Objetivo

Aplicar princípios de construção enxuta (Lean Construction) para redução de desperdícios, aumento de produtividade e melhoria contínua.

## Princípios Lean

### 1. Os 7 Desperdícios

1. **Superprodução**: Produzir mais que o necessário
2. **Espera**: Tempo ocioso de mão de obra/equipamentos
3. **Transporte**: Movimentação desnecessária de materiais
4. **Processamento**: Etapas que não agregam valor
5. **Estoque**: Materiais parados
6. **Movimentação**: Deslocamentos desnecessários
7. **Defeitos**: Retrabalho

### 2. Last Planner System (LPS)

```
Níveis de planejamento:
1. Longo prazo (6 meses): Macro atividades
2. Médio prazo (6 semanas): Lookahead planning
3. Curto prazo (1 semana): Plano semanal

PPC (Percentual de Plano Concluído):
PPC = (Atividades concluídas / Atividades planejadas) × 100%

Meta: PPC > 80%
```

### 3. Pull Planning

- Planejamento reverso (da entrega para trás)
- Envolvimento de todas as equipes
- Comprometimento coletivo
